# Spam SMS Detection using Machine Learning

## Introduction

This project detects whether an SMS message is Spam or Ham using Machine Learning techniques.

## Dataset

spam.csv

## Machine Learning Models

- Naive Bayes
- Logistic Regression

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- WordCloud

## Features

- Data Cleaning
- Exploratory Data Analysis
- TF-IDF Vectorization
- Model Training
- ROC Curve
- Confusion Matrix
- Real Time SMS Prediction
